# Changelog

Tous les changements notables de ce projet seront documentés ici.

## [1.0.0] - 2026-02-24

### Ajouté
- Système de vote avec double chiffrement OpenPGP
- Dashboard administrateur complet
- Interface CO (Centre de Comptage)
- Interface DE (Centre de Dépouillement)
- Génération automatique de PDFs (M2 et Résultats)
- Système de notifications toast
- Gestion des candidats avec photos
- Export PDF des résultats

### Sécurité
- Clés PGP RSA 4096 bits
- Séparation stricte des responsabilités
- Linking_id cryptographique
