# Guide de contribution

Merci de contribuer à Vote Électronique Sécurisé ! 🎉

## Comment contribuer

1. Fork le projet
2. Créer une branche (`git checkout -b feature/NouvelleFonctionnalite`)
3. Commit (`git commit -m 'Ajout de NouvelleFonctionnalite'`)
4. Push (`git push origin feature/NouvelleFonctionnalite`)
5. Ouvrir une Pull Request

## Standards de code

- Python: PEP 8
- JavaScript: ESLint
- Commits: Messages clairs en français

## Tests

Tous les tests doivent passer avant une PR.
```bash
# Backend
python manage.py test

# Frontend
npm run test
```
